
# Introduction
It is a simple example of getting Solana's block and transaction detail using the `@solana/web3.js` library.


# Library
node v18  
express  
@solana/web3.js  
log4js  
dotenv  
morgan  


# Installation
```bash

npm install

docker build -t solana-explorer-backend .

docker run -d -p 3000:3000 solana-explorer-backend

```
